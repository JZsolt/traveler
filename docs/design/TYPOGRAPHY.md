# Traveler Typography

## Principle

Typography is the main design tool.

Traveler should look editorial and readable, not like a dense dashboard.

## Type scale

Display:
- 40px
- 48px on larger mobile/tablet
- font-weight: 800
- letter-spacing: -0.04em
- use for trip titles only

H1:
- 32px
- font-weight: 800
- letter-spacing: -0.035em

H2:
- 24px
- font-weight: 700
- letter-spacing: -0.03em

H3:
- 18px
- font-weight: 700
- letter-spacing: -0.02em

Body:
- 16px
- line-height: 1.6

Small:
- 14px
- line-height: 1.5

Caption:
- 12px
- font-weight: 600
- letter-spacing: 0.06em
- uppercase only for section labels

## Usage

Trip title:
- Display

Section titles:
- Caption uppercase for label
- H2 for major section heading

Timeline item title:
- H3

Timeline description:
- Small / muted

Metadata:
- Caption or Small / muted

## Avoid

- many font sizes in one component
- tiny text for important content
- uppercase for long text
- center-aligned long content
