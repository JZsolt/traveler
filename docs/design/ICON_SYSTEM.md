# Traveler Icon System

## Library

Use Lucide icons only.

Do not use:
- emoji as interactive UI icons (buttons, navigation)
- mixed icon packs
- filled icons mixed with outline icons
- random SVGs unless they are part of branding

Emoji is acceptable as decorative content (section titles, trip flags).

## Default icon sizes

- Navigation icons: 20px
- Inline icons: 16px
- Card/section icons: 20px
- Empty state icons: 32px
- Hero decorative icon: rarely used, max 40px

## Stroke

Use Lucide default stroke width unless there is a strong reason not to.

## AI icon rule

Sparkles should not appear everywhere.

Use it only for:
- AI Assist button
- AI edit panel
- AI loading state

Do not use sparkles on every card.
